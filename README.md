## VoltaForth X

An old robot idea as a platform for new developments, using [Volatco](https://volatco.tech) and [polyForth](/docs/polyForth.pdf).

![image](/images/VoltaForthX.png)

### Background

A 1983-era commercial manufacture of Heiserman's [Rodney](https://github.com/cartheur/aiventure-rodney), seen in action [here](https://www.youtube.com/watch?v=YWzSIUBbF_E).
